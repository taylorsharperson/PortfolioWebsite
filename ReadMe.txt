# PortfolioWebsite

This is the personal portfolio website of **Taylor Sharperson**, a Computer Information Systems student at Florida Agricultural and Mechanical University. The site showcases academic achievements, projects, community service, and ways to contact Taylor.

---

## Project Structure

- **index.html** – The **Home Page**  
  Introduces Taylor, provides a brief personal background, and highlights involvement in programs such as Black Girls Code. Includes images, audio, and links to external resources.

- **hbcu.html** – **HBCU Life Page**  
  Details Taylor's experience attending Florida A&M University. Highlights campus life, student organizations, events, and photos showcasing personal experiences on campus.

- **service.html** – **Community Service Page**  
  Showcases volunteer work and community involvement. Includes details on participation in organizations such as Homeless Solutions and Table of Hope, along with images from service events.

- **projects.html** – **Projects & Internship Experience Page**  
  Highlights Taylor’s academic and technical projects, including programming projects, internship experience at Eli Lilly, and competitions. Contains interactive elements like videos and Q&A sections.

- **contact.html** – **Contact Page**  
  Provides contact information including email, phone, LinkedIn, and resume access. Features an embedded map showing Florida A&M University and iframes for media.

- **template.html** – **Template Page**  
  A reusable responsive page template for adding new sections or pages to the website. Includes a responsive grid layout and consistent header/footer.

- **css/styles.css** – **Stylesheet**  
  Contains all styling for the website including layout, typography, navigation, responsive design, and visual enhancements for tables, images, and forms.

- **scripts/script.js** – **JavaScript File**  
  Handles interactivity on the website such as Q&A sections and any dynamic elements.

- **images/** – Contains all images used throughout the website.

- **media/** – Contains video, audio, and PDF files embedded in the website (e.g., resume, project video, music).

---

## Features

- Fully **responsive layout** for mobile, tablet, and desktop.
- Integration of multimedia: images, videos, audio, and iframes.
- Interactive project section with questions and answers.
- Clean navigation across all pages.
- Modern fonts and styling for readability and aesthetics.
